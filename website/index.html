Quantum Advantage

The Challenge of Scale

With trillions of galaxies in the observable universe, each requiring analysis of gravitational lensing patterns, classical machine learning approaches face insurmountable computational barriers. The exponential growth in astronomical data demands quantum computing solutions that can:

  • Process high-dimensional feature spaces efficiently
  • Identify complex patterns in ellipticity measurements across millions of galaxies
  • Scale to handle the enormous datasets from next-generation telescopes
  • Provide faster inference for real-time dark matter mapping

This project demonstrates how quantum machine learning can tackle these challenges, processing galaxy data that would be computationally prohibitive for classical systems.

How It Works

01

Data Generation

We simulate realistic galaxy ellipticity measurements by combining:

  • Intrinsic Ellipticity: Random values between -0.05 and +0.05 representing natural galaxy shapes
  • Gravitational Shear: Strong shear (Î³ > 0.05) for lensed galaxies with dark matter, weak/zero shear for background galaxies

02

Quantum Model Architecture

Our Variational Quantum Classifier uses:

  • Feature Map: ZZFeatureMap with 2 repetitions to encode classical data into quantum states
  • Ansatz: RealAmplitudes with 3 repetitions for the variational circuit
  • Optimizer: COBYLA with 100 iterations
  • Loss Function: Cross-entropy for binary classification
03

Classification Process

The model performs binary classification:

About the Project

Research & Education

This project demonstrates how quantum machine learning can be applied to astrophysical problems, specifically the detection of dark matter through weak gravitational lensing. Dark matter halos cause gravitational shear, which distorts the shapes of background galaxies. By analyzing these distortions (measured as ellipticity), we can identify regions with dark matter concentrations.

The project uses Qiskit's Variational Quantum Classifier (VQC) to process galaxy ellipticity measurements and classify them based on dark matter presence. This is a research/educational project demonstrating quantum machine learning applications in astrophysics.
